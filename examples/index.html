üéÆ Getting Started

Welcome to Retro UI! This library provides pixel-perfect retro gaming components that work without any build step.

Just include the CSS and optional JavaScript files, and you're ready to go!

üì¶ Installation

Simply download the files and include them in your HTML:

<link rel="stylesheet" href="retro-ui.css">
<script src="retro-ui.js"></script>
üé® Theming

Customize the look using CSS custom properties:

:root {
  --retro-primary-bg: #your-color;
  --retro-primary-text: #your-text-color;
}
